# تنظیمات Claude Code

برای زمینه کامل پروژه — معماری، قراردادها، پشته فناوری و قوانین — فایل `AGENTS.md` را بخوانید.

---

## رفتار اختصاصی Claude Code

- هنگام فشرده‌سازی context، لیست کامل فایل‌های تغییریافته را حفظ کن
- برای وظایف پژوهشی از زیر-ایجنت‌ها استفاده کن، نه بررسی inline
- از مهارت‌های موجود در `.claude/skills/` برای انواع وظایف تکرارشونده استفاده کن
- هنگام تغییر به وظیفه‌ای کاملاً متفاوت، یک نشست جدید شروع کن

## مسیریابی زیر-ایجنت

- وظایف پژوهشی ← `.claude/skills/research.md`
