AI Agent Project Structure

Build once. Let every agent understand it. A tool-agnostic scaffold for Claude Code and OpenAI Codex.

The Three-Layer Architecture
📚

① Shared Knowledge

Docs, references, templates, and core logic that every tool can read. Written once, used everywhere.

/docs /scripts /templates
⚙️

② Workflows (Skills)

Reusable task guides and how-tos. Mirrored so any agent can find them regardless of tool.

.claude/skills/ .agents/
🔧

③ Tool Config

Settings unique to each agent. API keys, permission hints, model preferences. Kept isolated.

.claude/ .codex/

The Session Handoff Trick
1

Ask for a Summary

Tell the current agent: "Generate a session summary with current state, modified files, and next steps."

2

Copy the Output

The agent produces a structured handoff document. Copy it to your clipboard.

3

Paste into New Agent

Open the other agent, paste the summary as your first message. Work continues — no context lost.
